## 1. 产品概述

ESP32-S3-BOX-3 是乐鑫（Espressif）推出的新一代全开源 AIoT（人工智能物联网）开发套件，基于功能强大的 ESP32-S3 Wi-Fi + Bluetooth 5 (LE) SoC 构建。它面向 AIoT、边缘 AI（Edge AI）和工业物联网（IIoT）领域，拥有紧凑美观的外壳以及丰富的功能配件和转接配件，让开发者可以轻松定制和扩展功能。

ESP32-S3-BOX-3 出厂固件支持中文和英文离线语音唤醒和命令词识别。借助乐鑫可靠的软件解决方案（如 ESP-BOX、ESP-SR、ESP-RainMaker 和 ESP-Matter），无论是在原型开发新项目还是构建复杂的物联网系统，ESP32-S3-BOX-3 均能提供完美的形态和功能平衡。

### 核心亮点

- **出色的 AI 性能**：搭载 ESP32-S3 SoC，支持 AI 加速指令（向量指令），能够运行乐鑫自研语音识别框架 ESP-SR，为用户提供离线 AI 语音助手功能。
- **离线 + 在线 AI**：支持远场语音交互、连续识别和唤醒打断，可识别超过 200 个定制命令词；还可集成 OpenAI ChatGPT、百度文心一言等生成式 AI 平台，打造在线 AI 聊天机器人。
- **配件丰富，扩展灵活**：板载显示屏、触摸面板、双麦克风、扬声器、Type-C 端口，以及支持硬件扩展的高密度 PCIe 连接器和 Pmod 兼容接口。
- **完全开源**：硬件设计、固件代码、移动应用 APP 全部开源。

| 无线连接 | Wi-Fi (802.11 b/g/n) + Bluetooth 5 (LE) |

## 3. 显示屏

ESP32-S3-BOX-3 板载一块 2.4 英寸 SPI 接口的彩色 LCD 显示屏，具有以下特性：

| 参数 | 规格 |
|------|------|
| 屏幕尺寸 | 2.4 英寸 |
| 分辨率 | 320 x 240 (QVGA) |
| 接口 | SPI |
| LCD 驱动 IC | ILI9342 |
| 背光 | 可软件调节 |

该显示屏配合 LVGL（Light and Versatile Graphics Library）图形库使用，可以实现丰富的图形用户界面（GUI）。出厂固件已在显示屏上实现了多页面的交互式主界面，包括传感器监控、设备控制、网络设置、媒体播放器等功能页面。

## 4. 触摸屏

ESP32-S3-BOX-3 的显示屏配备了电容式触摸面板，支持多点触控：

| 参数 | 规格 |
|------|------|
| 触摸类型 | 电容式触摸 |
| 触摸驱动 IC | TT21100 / GT911 |
| 接口 | I2C |
| 支持手势 | 点击、滑动、长按等 |

触摸屏下方还设有一块红色圆圈触摸区域，用作自定义的返回按键。该区域允许用户根据自己的喜好配置额外的触摸功能。

开发板上还设有硬件静音按钮，位于设备顶部。按下该按钮可关闭语音唤醒和识别功能，再次按下即可恢复语音功能。

## 5. 麦克风

ESP32-S3-BOX-3 配备双数字麦克风阵列，专为远场语音拾音设计：

| 参数 | 规格 |
|------|------|
| 麦克风数量 | 2 个（立体声阵列） |
| 类型 | MEMS 数字麦克风 |
| 通信接口 | I2S |
| 音频 ADC | ES7210（4 通道高性能音频 ADC） |
| 特性 | 支持声学回声消除（AEC） |

双麦克风阵列配合乐鑫声学前端（ESP-AFE）算法，可实现：

- **远场语音拾音**：有效拾音距离可达 3-5 米
- **波束成形**：定向拾取目标方向的语音信号
- **降噪处理**：抑制环境噪声和回声
- **语音唤醒与识别**：支持连续识别和唤醒打断

## 6. 扬声器与音频系统

### 6.1 扬声器

ESP32-S3-BOX-3 板载一个扬声器，用于音频输出和语音交互反馈。

### 6.2 音频编解码芯片

| 组件 | 型号 | 说明 |
|------|------|------|
| 音频编解码器 | ES8311 | 低功耗单声道音频编解码器，包含单通道 ADC、单通道 DAC、低噪声前置放大器、耳机驱动器等 |
| 音频功率放大器 | NS4150 | 低 EMI、3W 单声道 D 类音频功率放大器 |

音频系统通过 I2S 和 I2C 总线与 ESP32-S3-WROOM-1 模组连接，提供独立于应用程序的硬件音频处理能力。出厂固件的 `Media Player` 功能页面支持音乐播放和系统音量调节。

## 8. 套件内容与变体

ESP32-S3-BOX-3 提供两款不同配置的套件：

### 8.1 ESP32-S3-BOX-3 标准版套件

| 组件 | 说明 |
|------|------|
| ESP32-S3-BOX-3 | 独立工作的主模块 |
| ESP32-S3-BOX-3-DOCK | 功能配件，可作为主机的座子 |
| ESP32-S3-BOX-3-SENSOR | 功能配件，展示传感器应用 |
| ESP32-S3-BOX-3-BRACKET | 转接配件，用于将主机安装在其他设备上 |
| ESP32-S3-BOX-3-BREAD | 转接配件，方便主机与标准面包板连接 |
| USB-C 数据线 | 供电和调试 |
| RGB LED 模块和杜邦线 | 测试用 |

### 8.2 ESP32-S3-BOX-3B 基础版套件

| 组件 | 说明 |
|------|------|
| ESP32-S3-BOX-3 | 独立工作的主模块 |
| ESP32-S3-BOX-3-DOCK | 功能配件，可作为主机的座子 |
| USB-C 数据线 | 供电和调试 |
| RGB LED 模块和杜邦线 | 测试用 |

> **区别说明**：标准版配备四款蓝色配件，基础版仅提供一款配件（DOCK），适用于不需要太多额外硬件的初学者。

## 9. 快速入门

### 9.1 给设备供电

1. 使用包装中提供的 USB-C 数据线连接设备与电源（5V 电源适配器或电脑 USB 端口）。
2. 设备启动后，屏幕上会显示 Espressif logo 的开机动画。

### 9.2 开始体验

设备上电后，系统将引导您完成以下初始化步骤：

1. **阅读按键引导页**：了解 BOX 开发板的按键功能，随后点按 `Next` 进入下一页。
2. **阅读语音助手说明**：了解 AI 语音控制功能，点按 `OK Let's Go` 进入主页。

### 9.3 主页功能

主页包含六大功能模块，左右滑动可在选项之间切换：

| 功能模块 | 说明 |
|----------|------|
| Sensor Monitor | 传感器数据监控（温湿度等） |
| Device Control | 设备控制（如灯光开关） |
| Network | 网络设置与连接 |
| Media Player | 音乐播放与音量调节 |
| Help | 帮助信息 |
| About Us | 关于信息 |

### 9.4 语音交互

设备出厂支持中英文离线语音交互：

1. 说出唤醒词唤醒设备。
2. 唤醒后说出命令词，设备将执行相应操作。
3. 使用设备顶部的静音按钮可随时关闭/恢复语音功能。

> 通过 ESP-BOX 手机 APP，您可以创建个性化的命令词来控制智能设备，实现可重配置的 AI 语音交互。

## 10. 软件开发

### 10.1 开发框架

ESP32-S3-BOX-3 的开发基于以下软件框架：

| 框架/SDK | 说明 |
|----------|------|
| ESP-IDF | 乐鑫官方 IoT 开发框架，基于 FreeRTOS |
| ESP-SR | 乐鑫语音识别框架，支持离线语音唤醒和命令词识别 |
| ESP-ADF | 乐鑫音频开发框架 |
| ESP-DL | 深度学习开发库，支持神经网络模型部署 |
| ESP-Matter | Matter 协议实现，支持跨平台智能家居控制 |
| ESP RainMaker | 乐鑫 AIoT 云平台，支持 OTA 升级和远程控制 |
| LVGL | 轻量级图形库，用于构建用户界面 |

### 10.2 BSP 支持

乐鑫为 ESP32-S3-BOX-3 提供了官方板级支持包（BSP），可通过 ESP Component Registry 获取：

- 组件名称：`espressif/esp-box-3`
- 支持：显示屏、触摸屏、音频（ES8311/ES7210）、按键、SD 卡、IMU、温湿度传感器等

### 10.3 示例代码

ESP-BOX 仓库提供了丰富的示例代码：

- **ChatGPT Demo**：集成 OpenAI ChatGPT 的在线 AI 聊天机器人演示
- **Voice Assistant**：离线语音助手演示
- **HMI 演示**：基于 LVGL 的人机交互界面
- **Matter 网关**：智能家居多协议网关
- **传感器应用**：环境监测演示

### 10.4 烧录固件

```bash
# 设置目标芯片
idf.py set-target esp32s3

# 编译项目
idf.py build

# 烧录固件
idf.py -p <PORT> flash

# 监视串口输出
idf.py -p <PORT> monitor
```
